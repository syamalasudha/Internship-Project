# 🎨 Advanced Portfolio Website

A modern, responsive developer portfolio built using **HTML5**, **CSS3**, and **JavaScript**. The project focuses on advanced CSS concepts such as **CSS Grid**, **Flexbox**, **CSS Variables**, **Animations**, **Glassmorphism**, and **Responsive Design** while following the **BEM (Block Element Modifier)** naming methodology.

---

# 📌 Project Overview

This portfolio showcases personal information, technical skills, featured projects, professional journey, and contact details in a clean and interactive interface. Instead of using a traditional navigation bar, the homepage features a modern **Bento Grid** layout that provides quick access to different sections.

The design emphasizes smooth user interaction, accessibility, responsive layouts, and visually appealing animations.

# 🎯 CSS Concepts Implemented

## CSS Grid

Used for:

* Bento Grid Homepage
* Skills Layout
* Project Gallery
* Footer Layout
* Contact Section

## Flexbox

Used for:

* Buttons
* Hero Content
* Footer
* Social Icons
* Contact Details

## CSS Variables

Implemented using:

```
:root {
    --primary
    --secondary
    --bg
    --surface
    --text
}
```

This allows easy theme customization and supports light/dark mode.

## Animations

Implemented using CSS Keyframes:

* Fade Up
* Fade Left
* Fade Right
* Zoom In
* Floating Blobs
* Pulse
* Bounce
* Shine Effect
* Hover Animations
* Scroll Reveal

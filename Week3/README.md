## 🧠 Basic Concepts Used in This Project

This project is built using core frontend technologies: HTML, CSS, and JavaScript.

---

## 🌐 HTML (Structure)
HTML is used to create the structure of the web page.

### We used HTML for:
- Creating forms (name, email, message)
- Adding buttons for actions
- Displaying images in gallery
- Creating sections and layout
- Adding modal popup structure

---

## 🎨 CSS (Design & Styling)
CSS is used to design and style the web page.

### We used CSS for:
- Background colors and gradients
- Card-style layout design
- Button styling and hover effects
- Image gallery layout
- Dark mode styling
- Modal popup styling
- Smooth transitions and animations

---

## ⚙️ JavaScript (Functionality)
JavaScript is used to make the website interactive.

---

## 🧠 Key JavaScript Concepts Used

### 1. DOM Manipulation
Used to change HTML content dynamically like:
- Updating text
- Showing messages
- Changing UI based on user actions

---

### 2. Event Listeners
Used to detect user actions such as:
- Click events (buttons, images)
- Form submit events
- Page load events

---

### 3. Form Validation
Used to check user input before submission:
- Name should not be empty
- Email must be valid
- Message must have minimum length
- Error messages are shown if invalid

---

### 4. Reusable Functions
Functions are used to avoid repeating code:
- Showing errors
- Clearing error messages

---

### 5. Image Gallery (Modal Feature)
- Clicking an image opens it in a popup
- Displays large version of image
- Clicking outside or close button closes it

---

### 6. Local Storage
Used to store data in browser:
- Saves user name
- Saves theme preference (dark/light mode)
- Data remains after page refresh

---

### 7. Conditional Statements
Used for decision making:
- If input is empty → show error
- If input is valid → show success message

# 🍽️ Delicious Bites - Restaurant Website

A modern, fully responsive restaurant website built using **HTML5, CSS3, and JavaScript**. This website showcases the restaurant's menu, story, contact information, and reservation form with a clean and attractive user interface.

---

## 📖 Project Description

**Delicious Bites** is a responsive restaurant website designed to provide visitors with an engaging online dining experience. Customers can browse the menu, learn about the restaurant, view featured dishes, and reserve a table through the contact page.

The website is built with semantic HTML, modern CSS, and JavaScript while following responsive web design principles.

---

## ✨ Features

* 🍔 Modern and attractive restaurant homepage
* 📱 Fully responsive design (Desktop, Tablet & Mobile)
* 🧭 Sticky navigation bar with mobile hamburger menu
* 🍕 Interactive food menu with pricing
* 👨‍🍳 About Us page with restaurant story
* 📞 Contact page with reservation form
* ✅ JavaScript form validation
* 🖼️ Restaurant gallery
* ⭐ Customer testimonials
* ⬆️ Scroll-to-top button
* ✨ Smooth animations and hover effects
* 📍 Embedded Google Map
* 🎨 Professional UI with modern color scheme

## 🚀 How to Run the Project

1. Download or clone the repository.
2. Extract the project folder (if downloaded as a ZIP).
3. Open the **Delicious-Bites** folder.
4. Double-click `index.html` or open it using **Live Server** in Visual Studio Code.
5. Browse the website and navigate through all the pages.
